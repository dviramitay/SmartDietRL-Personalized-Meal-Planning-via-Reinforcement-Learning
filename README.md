# SmartDietRL-Personalized-Meal-Planning-via-Reinforcement-Learning


This project uses a Q-Learning agent to construct an optimal daily meal plan consisting of four meals (breakfast, lunch, dinner, and snack) while optimizing for nutritional goals such as protein intake, calorie balance, and satiety. The agent learns to select meals from a predefined set, receiving rewards and penalties based on how well the selected meals meet dietary targets.

Main Features:

Custom MealEnv environment simulating daily nutritional decisions

Reward function balances protein goals, caloric range, and satiety

Penalization for excessive fat intake or failing nutritional goals

Clear visualization of macronutrient distribution and learning progression

Evaluation and analysis tools for performance breakdown

 Trained over 40,000 episodes using epsilon-greedy exploration
 Targeted for weight-loss with smart nutrient trade-offs
 Includes visual analytics (bar/pie charts, reward trends)
